# ⚡️ SSAFY Sandbox


## 📝 프로젝트 소개
> SSAFY Sandbox 프로젝트는 SSAFY 12기 서울 15반의 백엔드 개발자를 희망하는 교육생들의 협업 역량 향상을 위해 기획되었습니다. 
> 
> 백엔드 개발자들이 구현한 기능을 직접 UI를 통해 확인하고, 프론트엔드 개발자들과의 실질적인 협업 경험을 쌓을 수 있도록 설계된 프로젝트입니다.
